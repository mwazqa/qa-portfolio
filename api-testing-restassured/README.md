# API Testing - RestAssured

## Opis
Folder zawiera automatyczne testy API napisane w Javie z użyciem RestAssured. Zawiera modele danych, klasy pomocnicze oraz testy.

## Struktura folderów
- **src/main/java/models/** – modele danych
- **src/main/java/utils/** – klasy pomocnicze
- **src/test/java/api/** – testy API
- **src/test/java/base/** – klasy bazowe i konfiguracje
- **allure-results/** – wyniki raportów Allure
- **target/** – pliki kompilowane (ignorowane w Git)

# API Testing - RestAssured

## Overview
This folder contains automated API tests written in Java using RestAssured. It includes data models, utility classes, and test classes.

## Folder Structure
- **src/main/java/models/** – data models
- **src/main/java/utils/** – utility classes
- **src/test/java/api/** – API tests
- **src/test/java/base/** – base classes and configurations
- **allure-results/** – Allure report results
- **target/** – compiled files (ignored by Git)
