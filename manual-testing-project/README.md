# Manual Testing Project

## Opis
Folder zawiera zasoby do testów manualnych, w tym plany testów, przypadki testowe, zgłoszenia błędów i listy kontrolne.

## Struktura folderów
- **TestPlan/**: dokumenty planu testów
- **TestCases/**: przypadki testowe
- **BugReports/**: zgłoszenia błędów
- **Checklists/**: listy kontrolne do weryfikacji
