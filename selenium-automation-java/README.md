# Selenium Automation - Java

## Opis
Folder zawiera automatyczne testy UI napisane w Javie przy użyciu Selenium.  
Zawiera Page Object Model, testy UI, sterowniki przeglądarek i konfiguracje CI/CD.

## Struktura folderów
- **src/main/java/pages/** – klasy Page Object reprezentujące strony
- **src/test/java/ui/** – testy automatyczne UI
- **drivers/** – sterowniki przeglądarek
- **allure-results/** – wyniki raportów Allure
- **.github/workflows/** – konfiguracja GitHub Actions
- **target/** – pliki kompilowane (ignorowane w Git)

# Selenium Automation - Java

## Overview
This folder contains automated UI tests written in Java using Selenium.  
It includes Page Object Model, UI test classes, browser drivers, and CI/CD configurations.

## Folder Structure
- **src/main/java/pages/** – Page Object classes representing pages
- **src/test/java/ui/** – automated UI tests
- **drivers/** – browser drivers
- **allure-results/** – Allure report results
- **.github/workflows/** – GitHub Actions configuration
- **target/** – compiled files (ignored by Git)
